# TicTacToeTui

## Simple Tic Tac Toe Tui game written in rust


```
  1 | 2 | 3
  -----------
  4 | 5 | 6
  -----------
  7 | 8 | 9
```
diagram for playing


## To play:
- Have cargo and git installed

- clone this repository
```
git clone https://github.com/gamcd/TicTacToeTui.git
```
- Run the project
```
cargo run
```
